FissileKit

Guia de uso — descargas, conversiones y editor de imagenes

v1.2.0 · Toolkit multimedia

1 YouTube — un link

  1. Pega un enlace de YouTube en el campo principal.
  2. Elige Video o Audio y la calidad.
  3. Selecciona la carpeta de salida si quieres cambiarla.
  4. Pulsa Descargar.

2 YouTube — lote

  1. Agrega enlaces con Agregar al lote o Cargar .txt (un link por linea).
  2. Activa el toggle si quieres usar la tecla global para mandar al lote.
  3. Pulsa Descargar lote — mismo formato y calidad para todos.

3 Imagenes

  1. Elige fuente: Pexels (requiere API key) o Flickr Public.
  2. Llena la lista con terminos o enlaces y pulsa Buscar imagenes.
  3. Pega imagenes con Pegar ahora o la tecla global.
  4. En el lote usa Otra o Quitar para ajustar resultados.
  5. Pulsa Descargar lote cuando este listo.

Pexels: configura tu API key en Configuracion (engranaje en la barra superior).

4 Conversion

  1. En la pestaña Conversion, pulsa Traer Archivo o Elegir lote....
  2. Elige el formato de destino (imagen, video o audio segun el archivo).
  3. Para imagen → video: ajusta duracion y calidad.
  4. Para imagen → audio: genera un sonido a partir de los colores de la imagen.
  5. Pulsa Convertir. Usa Ver lote para revisar varios archivos.

FFmpeg amplia los formatos disponibles (MP4, MP3, SVG, espectrogramas, etc.).

5 Editor — lo basico

  1. Abre la pestaña Editor. A la izquierda veras el historial de tu carpeta FissileKit.
  2. Selecciona una imagen en el arbol o haz clic en el lienzo para abrir un archivo.
  3. Elige una herramienta en la barra de iconos. Aparece una sub-barra con opciones.
  4. Guardar en la sub-barra: aplica cambios en la sesion y vuelve a la barra principal (no guarda archivo).
  5. Salir: abandona la herramienta (confirma si hay cambios sin aplicar).
  6. Terminar (abajo): exporta el resultado a disco (PNG si hay transparencia).

6 Editor — herramientas

Recorte

  • Esquinas: arrastra los puntos del marco.
  • Arrastrar: dibuja el area con el raton.
  • Proporciones: libre, 1:1, 9:16 o 16:9.
  • Fuera del recorte queda transparente (mascara).

Rotar

  • Rueda del raton, botones de 90° o campo de grados + Enter.
  • Volteo horizontal y vertical.

Redimensionar

  • Escalar: cambia el tamano de la imagen.
  • Lienzo: coloca la imagen dentro de un lienzo fijo (HD, Full HD, 4K). Arrastra para mover; rueda para escalar.

Dibujar

  • Lapiz, Borrador (solo trazos dibujados), Gotero y Cubeta.
  • Ajusta tamano, opacidad y tolerancia (cubeta).

Borrador PNG

  • Herramienta aparte en la barra principal: borra la imagen y deja transparencia (exporta en PNG).
  • Manual: pincel que borra pixeles de la imagen.
  • Color total: un clic elimina ese color en toda la imagen (tolerancia).
  • Zona: un clic elimina la zona conectada del color.
  • Magico: primer clic elige color; arrastra para borrar zonas similares.
  • Reparar: pincel que restaura la imagen original en esa zona.

Formas

  • Linea, curva (clics + doble clic para terminar), rectangulo, ovalo, circulo, triangulo, pentagono, hexagono, estrella.
  • Relleno y borde opcionales, con color y grosor.
  • Arrastra para definir tamano; un clic sin arrastrar crea una figura proporcional.
  • Circulo desde el centro; el resto como caja (esquina a esquina).

Texto

  • Clic en la imagen: aparece el cuadro con cursor para escribir.
  • Fuente, tamano (min. 24 px), negrita, cursiva, subrayado, tachado y color.
  • Borde opcional (activar y elegir grosor/color).
  • Arrastra el texto para moverlo; esquinas para escalar.
  • Tras escribir: un clic fuera deselecciona; otro clic en vacio crea texto nuevo.
  • Doble clic en un texto existente para editarlo de nuevo.

7 Tecla global

  1. Abre Configuracion en la barra superior.
  2. En Tecla rapida global, escribe una tecla o combinacion.
  3. Ejemplos: x, f8, ctrl+shift+x.
  4. Pulsa Guardar.

La tecla no se activa mientras escribes en un campo de texto.

9 Consejos

  • Para una sola letra como atajo global, elige una que no uses al escribir.
  • Si Pexels no encuentra algo, prueba Flickr Public.
  • Con ffmpeg instalado, video, audio y conversiones avanzadas funcionan mejor.
  • Tema e idioma se cambian en Configuracion.
  • En el editor, usa PNG al exportar si trabajaste con transparencia o capas de dibujo.
  • La carpeta del editor por defecto es Downloads/FissileKit; puedes cambiarla en Configuracion.
