Instructivo de uso

Descargador Multimedia

Esta version del instructivo va en texto para que sea rapida y clara. Si quieres una version con fotos, mandame capturas de las pantallas clave y la armamos.

1. YouTube

  1. Pega un link de YouTube en el campo principal.
  2. Elige Video o Audio.
  3. Elige la calidad.
  4. Selecciona la carpeta de salida si quieres cambiarla.
  5. Pulsa Descargar para un solo link.

2. Lote de YouTube

  1. Pega un link y usa Agregar al lote.
  2. Repite con todos los links que quieras.
  3. Tambien puedes usar Cargar .txt con un link por linea.
  4. Pulsa Descargar lote.
  5. Todo el lote usa el mismo formato y la misma calidad.

3. Imagenes

  1. Elige la fuente: Pexels o Flickr Public.
  2. Llena la lista con terminos o links.
  3. Pulsa Buscar imagenes para agregar resultados al lote.
  4. Si copias una imagen, puedes usar Pegar ahora.
  5. En el lote puedes quitar una imagen o cambiarla con Otra.
  6. Cuando el lote quede bien, pulsa Descargar lote.

4. Tecla global

  1. Pulsa el boton ....
  2. En Tecla rapida global, escribe una tecla o combinacion.
  3. Ejemplos: x, f8, ctrl+shift+x.
  4. Pulsa Guardar.

5. Toggles del lote

  • En YouTube, el toggle manda texto o links al lote de YouTube con la tecla global.
  • En Imagenes, el toggle manda imagenes o texto al lote/lista con la tecla global.
  • Si el toggle esta apagado, la tecla global no manda nada a ese lote.
  • La app intenta ignorar la tecla global cuando detecta que estas escribiendo en un campo de texto.

6. Botones utiles

  • Abrir carpeta: abre la carpeta de salida actual.
  • Limpiar lote: vacia la cola actual.
  • i: abre este instructivo.

7. Sugerencias

  • Si vas a usar una sola letra como tecla global, prueba una que no uses mucho.
  • Si quieres algo mas seguro, usa una combinacion como ctrl+shift+x.
  • Si Pexels no encuentra algo, cambia a Flickr Public.
  • Si tienes ffmpeg, el resultado final suele salir mejor.
